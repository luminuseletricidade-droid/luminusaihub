# Fluxo de Autenticacao

## Diagrama de Sequencia - Login

```
┌─────────┐          ┌─────────┐          ┌─────────┐          ┌─────────┐
│ Usuario │          │Frontend │          │ Backend │          │   DB    │
└────┬────┘          └────┬────┘          └────┬────┘          └────┬────┘
     │                    │                    │                    │
     │  1. Acessa /auth   │                    │                    │
     │───────────────────>│                    │                    │
     │                    │                    │                    │
     │  2. Exibe form     │                    │                    │
     │<───────────────────│                    │                    │
     │                    │                    │                    │
     │  3. Preenche       │                    │                    │
     │     email/senha    │                    │                    │
     │───────────────────>│                    │                    │
     │                    │                    │                    │
     │                    │  4. POST /api/auth/signin              │
     │                    │───────────────────>│                    │
     │                    │                    │                    │
     │                    │                    │  5. SELECT user   │
     │                    │                    │───────────────────>│
     │                    │                    │                    │
     │                    │                    │  6. user data     │
     │                    │                    │<───────────────────│
     │                    │                    │                    │
     │                    │                    │  7. Valida senha   │
     │                    │                    │     (bcrypt)       │
     │                    │                    │                    │
     │                    │                    │  8. Gera JWT       │
     │                    │                    │                    │
     │                    │  9. { user, token }│                    │
     │                    │<───────────────────│                    │
     │                    │                    │                    │
     │                    │  10. Salva token   │                    │
     │                    │      localStorage  │                    │
     │                    │                    │                    │
     │  11. Redirect      │                    │                    │
     │      /app/dashboard│                    │                    │
     │<───────────────────│                    │                    │
     │                    │                    │                    │
```

## Verificacao de Token (Middleware)

```
┌─────────────────────────────────────────────────────────────────┐
│                         Requisicao HTTP                          │
└─────────────────────────────────┬───────────────────────────────┘
                                  │
                                  ▼
                    ┌─────────────────────────┐
                    │ Header Authorization?    │
                    └───────────┬─────────────┘
                                │
                    ┌───────────┴───────────┐
                    │                       │
                    ▼                       ▼
               ┌────────┐             ┌──────────┐
               │  Sim   │             │   Nao    │
               └────┬───┘             └────┬─────┘
                    │                      │
                    ▼                      ▼
          ┌─────────────────┐        ┌──────────────┐
          │ Extrai Bearer   │        │ 401          │
          │ Token           │        │ Unauthorized │
          └────────┬────────┘        └──────────────┘
                   │
                   ▼
          ┌─────────────────┐
          │ Valida JWT      │
          │ (secret, exp)   │
          └────────┬────────┘
                   │
         ┌─────────┴─────────┐
         │                   │
         ▼                   ▼
    ┌─────────┐         ┌─────────┐
    │ Valido  │         │Invalido │
    └────┬────┘         └────┬────┘
         │                   │
         ▼                   ▼
┌──────────────────┐   ┌──────────────┐
│ Extrai user_id   │   │ 401          │
│ do payload       │   │ Token        │
│                  │   │ Invalid      │
│ Continua         │   └──────────────┘
│ requisicao       │
└──────────────────┘
```

```json
  },
  "payload": {
    "sub": "user_uuid",
    "email": "user@email.com",
    "name": "Nome Usuario",
    "role": "user",
    "iat": 1704067200,
    "exp": 1704153600
  },
  "signature": "..."
}
```

## Rotas Protegidas

```
┌─────────────────────────────────────────────────────────────────┐
│                      React Router                                │
└─────────────────────────────────┬───────────────────────────────┘
                                  │
                                  ▼
                    ┌─────────────────────────┐
                    │   ProtectedRoute        │
                    │   Component             │
                    └───────────┬─────────────┘
                                │
                    ┌───────────┴───────────┐
                    │ useAuth()             │
                    │ isAuthenticated?      │
                    └───────────┬───────────┘
                                │
                    ┌───────────┴───────────┐
                    │                       │
                    ▼                       ▼
               ┌────────┐             ┌──────────┐
               │  true  │             │  false   │
               └────┬───┘             └────┬─────┘
                    │                      │
                    ▼                      ▼
          ┌─────────────────┐        ┌──────────────┐
          │ Renderiza       │        │ Redirect     │
          │ children        │        │ /auth        │
          └─────────────────┘        └──────────────┘
```
