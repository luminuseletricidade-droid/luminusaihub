================================================================================
TROUBLESHOOTING
================================================================================

PROBLEMA: "relation already exists"
SOLUÇÃO: Normal, IF NOT EXISTS deve prevenir erro. Pode ignorar.

PROBLEMA: "column already exists"  
SOLUÇÃO: Normal, IF NOT EXISTS deve prevenir erro. Pode ignorar.

PROBLEMA: "policy already exists"
SOLUÇÃO: Impossível, pois fazemos DROP POLICY IF EXISTS antes. Verifique logs.

PROBLEMA: Transaction abortada
SOLUÇÃO: Verifique o erro específico no log. Corrija e execute novamente.

PROBLEMA: Foreign key violation
SOLUÇÃO: Banco não está vazio? Limpe dados incompatíveis primeiro.

PROBLEMA: Out of memory
SOLUÇÃO: Execute em partes menores ou aumente recursos do servidor.

PROBLEMA: Timeout
SOLUÇÃO: Aumente timeout do Supabase ou execute em partes.
